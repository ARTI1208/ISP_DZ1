# ISP_DZ0

Quadratic equation solver

## Build && run
On linux, you can build using CMake
    
    cmake -S ./ -B out && cmake --build out
    
or by gcc 
    
    g++ -Wall -Wextra -Wfloat-equal -o out/executable src/*.cpp
    
and run generated executable
   
    out/executable
